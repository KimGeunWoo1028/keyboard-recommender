# Small Group Test Checklist

> **버전:** v1.0 - 2026-07-16 (KST)  
> **목적:** `www.keyboard-recommender.com`을 소수 인원에게 먼저 공개해 핵심 플로우를 검증할 때 사용할 운영 체크리스트  
> **범위:** 기능 추가 없이 현재 공개 URL에서 계정·설문·결과·저장·카탈로그 흐름을 확인  
> **관련:** `deployment-roadmap.md` · `remaining-work-phases.md` · `product-next-phase5-home-revisit.md`

---

## 이 문서의 목적

- 정식 production 분리 전, 제한된 인원으로 실제 사용 흐름을 먼저 검증한다.
- 치명적 오류 없이 핵심 플로우가 도는지 확인한다.
- `eval_events`와 `home.viewed` 표본을 쌓기 시작한다.
- 테스트 기간 동안은 Home revisit 제품 변경을 열지 않는다.

---

## 테스트 운영 원칙

- 테스트 대상 URL은 `https://www.keyboard-recommender.com`
- 기능 추가 배포 없이 현재 빌드 안정성 확인이 우선
- 버그 제보 채널은 1개로 통일
- 테스트 중 발견된 이슈는 재현 경로와 계정 상태를 같이 기록
- Home Dashboard / Login redirect / dual Hero는 열지 않음

## 참여자 테스트 체크리스트

### 1. 진입

- [ ] 홈 첫 진입 시 레이아웃 깨짐 없음
- [ ] 모바일/데스크톱 중 본인 사용 기기에서 첫 화면 확인 가능
- [ ] CTA 클릭 시 이동 정상

### 2. 계정

- [ ] 회원가입 가능
- [ ] 이메일 인증 메일 수신 가능
- [ ] 로그인 가능
- [ ] 새로고침 후 로그인 유지
- [ ] 로그아웃 가능

### 3. 추천 설문

- [ ] 설문 진입 가능
- [ ] 단계 이동 정상
- [ ] 답변 완료 후 결과 화면 진입 가능

### 4. 결과

- [ ] 결과 카드가 비정상적으로 비어 있지 않음
- [ ] 추천 설명 문구가 깨지지 않음
- [ ] Evidence/근거 영역 확인 가능
- [ ] 최근 결과 다시 보기 동작

### 5. 저장/마이페이지

- [ ] 결과 저장 가능
- [ ] 마이페이지 저장한 빌드에 반영
- [ ] 저장 후 다시 보기 가능

### 6. 카탈로그

- [ ] 카탈로그 진입 가능
- [ ] 탭 이동 가능
- [ ] 카드 이미지 로드 정상
- [ ] 스웨그키 링크 이동 정상
