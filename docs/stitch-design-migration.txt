Keyboard Recommender — Stitch(KeebSmith) 디자인 마이그레이션 로드맵
====================================================================

작성 목적
---------
Stitch 시안(stitch_keyboard_curator.zip)의 룩앤필만 입히고,
기존 기능·라우트·API·상태 로직은 100% 유지한다.

기준일: 2026-07-08
시안 경로 (원본): c:\Users\jeung\Downloads\stitch_keyboard_curator.zip
시안 구성:
  - cyber_artisan/DESIGN.md          … 디자인 토큰 (Cyber-Artisan)
  - keebsmith_modernized_1           … Recommended Builds (= /results)
  - keebsmith_modernized_2           … 랜딩/홈 (= /)
  - keebsmith_modernized_3           … 마이페이지 (= /mypage)
  - keebsmith_modernized_4           … Build Comparison (= 비교 워크스페이스)


핵심 원칙
---------
- 기능 100% 유지: 설문 → 추천 compute → 결과 → 북마크 → 마이페이지 → 카탈로그 → auth
- Stitch HTML을 페이지로 통째 교체하지 않는다. 기존 React 컴포넌트에 스타일·마크업만 이식한다.
- 시안에 있으나 백엔드에 없는 데이터(가격 $, Acoustic chart, Mass, Actuation curve,
  Credits, 알림 종 등)는 실기능처럼 넣지 않는다. 스킵하거나 순수 장식(비인터랙션)만.
- 카탈로그 5탭 · 스웨그키 shop_view/?idx= · responseContractRev · 쿠키 auth 규칙 유지
- 한국어 UI 유지 (시안 영문 카피는 톤만 참고, 라벨은 기존 한국어 우선)

--------------------
